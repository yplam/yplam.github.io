不能忽略 .htaccess 对性能的影响

  • In Linux Apache
  • /
  • Posted 2015-08-25

在Apache环境下使用 .htaccess 进行URL重写或许已经是一种非常常用的方式；然而，如果URL重写规则比较复杂，使用.htaccess的方式可能会影响到Apache的性能。
